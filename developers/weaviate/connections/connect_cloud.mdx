## Environment variables

Do not hard-code your API key in your client code. Consider passing the API key as an environment variable or using a similar secure coding technique.

To create an environment variable, add a line like this example to your shell configuration file.

```bash
export WEAVIATE_API_KEY="replaceThisPartWithYourWeaviateAPIKey"
```
